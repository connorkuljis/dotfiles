# Functionality

What is this code repository for?

- Deploying a consistent, system / user software development environment for linux based distributions.

Encompasses a number of packaging tools:
- APT (Advanced Packaging Tool) - The Debian Project, based on DPKG (Debian Package).
- DNF (Dandified YUM) - RedHat Project, based on RPM (RedHat Package Manager).
- NPM (Node Package Manager).
- Cargo - Rust package manager.
- Go tools - Retrieve and install remote packages with Go.

# Background

1. Software configuration management (SCM) - files that define software behaviour.
2. Infrastructure as Code (IaC) - automates provisioning 2 application software.

# Technology Stack
